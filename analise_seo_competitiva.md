## 2. PADRÃO IDENTIFICADO: DOMÍNIO DOS E-COMMERCES

### Por que E-commerces Ranqueiam Melhor?

**Fatores de Ranqueamento do Google:**

1. **Intenção Transacional**
   - Google identifica "argamassa baritada" como termo com intenção de compra
   - E-commerces atendem diretamente essa intenção
   - Usuários clicam mais em resultados com preços

2. **Sinais de Engajamento**
   - Tempo na página maior (navegação de produtos)
   - Taxa de clique elevada (CTR)
   - Baixa taxa de rejeição (bounce rate)

3. **Schema Markup de Produto**
   - Preços, avaliações, disponibilidade
   - Rich snippets atraem mais cliques
   - Google favorece resultados ricos

4. **Autoridade de Domínio**
   - Marketplaces têm DA altíssimo
   - Backlinks massivos
   - Histórico de confiança

## 4. GAP COMPETITIVO PRINCIPAL

**O Google favorece sites que:**
1. Mostram preços transparentes
2. Têm transações diretas
3. Possuem reviews de clientes
4. Carregam rapidamente (<2s)
5. Têm autoridade de domínio
6. Oferecem informação completa sobre produto

**Nosso site atual:**
- É um "intermediário" (lead capture)
- Não mostra preços
- Não tem transações
- Não tem reviews próprios

## 10. ESTRATÉGIA PARA SUPERAR E-COMMERCES

### Desafio Principal:
Como um site de lead capture pode ranquear acima de e-commerces gigantes?

### Solução Híbrida:

#### 1. Adicionar Elementos de E-commerce (sem vender)
- **Mostrar faixas de preço reais**: "A partir de R$ 24,99/saco"
- **Schema de Oferta**: Implementar AggregateOffer com múltiplos fornecedores
- **Indicadores de disponibilidade**: "Em estoque - Entrega 24h"
- **Comparador de preços**: Mostrar 3-5 fornecedores com preços

#### 2. Superar em Conteúdo Técnico
- Criar guia definitivo de argamassa baritada (5000+ palavras)
- Vídeos explicativos (YouTube + embed)
- Infográficos técnicos
- Calculadoras avançadas
- Comparadores de produtos

#### 3. Autoridade e Certificações
- Destacar parceria com fornecedores certificados CNEN
- Mostrar certificações dos parceiros
- Criar seção "Por que confiar"
- Adicionar depoimentos em vídeo

#### 4. Performance Técnica Superior
- Core Web Vitals perfeitos (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- Score 95+ no PageSpeed
- Imagens WebP otimizadas
- CSS/JS inline crítico
- Lazy loading

#### 5. Schema Markup Avançado
```json
{
  "@type": "Product",
  "name": "Argamassa Baritada",
  "offers": {
    "@type": "AggregateOffer",
    "lowPrice": "24.99",
    "highPrice": "34.50",
    "priceCurrency": "BRL",
    "offerCount": "5"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "127"
  }
}
```

#### 6. Conteúdo Diferenciado
- **Calculadora Avançada**: Mais precisa que concorrentes
- **Comparador de Fornecedores**: Transparência total
- **Guia de Aplicação**: Passo a passo com fotos
- **Normas e Certificações**: Explicação detalhada
- **FAQ Expandido**: 20+ perguntas

#### 7. SEO Local Agressivo
- Páginas por cidade otimizadas
- Schema LocalBusiness para cada região
- Reviews por localização
- Mapas de cobertura
